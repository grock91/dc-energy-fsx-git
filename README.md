# Data Center Energy Observatory — FutureScaleX

Interactive dashboard showing data center electricity consumption as a percentage of total grid load across regions, countries, and states — from 2000 to 2050.

## Features
- 3-level drill-down: Regions → Countries → States
- Animated time slider: 2000–2050
- Authoritative sources: IEA, EPRI, EIA, LBNL, CSO Ireland, Pew Research, Ember, Wood Mackenzie
- Visual forecast boundary (2026)

## Deployment
This project auto-deploys to GitHub Pages via GitHub Actions on push to `main`.

## Run Locally
```bash
npm install
npm run dev
```
